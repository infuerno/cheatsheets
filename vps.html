Maintenance

aptWrapper for common apt-get and apt-cache commands (man apt)
sudo apt updateResynchronize package index
sudo apt full-upgradeUpgrade packages and additionally install or remove dependencies as necessary
sudo apt cleanClean up package files
sudo apt autoremoveRemove autoinstalled packages which are no longer required
